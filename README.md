Project Overview

This project presents an end-to-end analysis of COVID-19 case and vaccination data using Python for data preprocessing and exploratory data analysis (EDA), followed by the development of interactive Tableau dashboards to visualize key pandemic trends across countries.
The objective of this project was to transform raw COVID-19 datasets into clean, structured, and analysis-ready data, uncover meaningful insights through statistical exploration, and communicate those insights through interactive business intelligence dashboards.

Data Sources

The project integrates multiple publicly available datasets containing:

Country-wise COVID-19 statistics
India state-wise COVID-19 case reports
State-wise vaccination data
Daily confirmed, recovered, deceased, and active cases


Data Cleaning & Preprocessing

The raw datasets contained inconsistencies, redundant information, and formatting differences that required extensive preprocessing before analysis.

The data preparation process included:

Imported multiple CSV datasets using Pandas.
Examined dataset structure, column data types, and missing values.
Removed duplicate records to maintain data integrity.
Identified and handled null or incomplete observations.
Standardized column names for consistency across datasets.
Converted numerical and date columns into appropriate data types.
Filtered unnecessary attributes to retain analysis-relevant features.
Verified data consistency across multiple datasets before merging.
Organized country-level and state-level datasets separately for efficient analysis.
Prepared clean, structured datasets suitable for visualization in Tableau.

The preprocessing pipeline ensured that the datasets were accurate, consistent, and optimized for downstream analytics and dashboard development.
